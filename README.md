# Smart Student Attendance System

> **Note:** *This is the project made by "ProjectMate" which help student to make projects.*

Welcome to the Smart Student Attendance System! This application is a full-stack web solution designed to help educational institutions track, manage, and analyze student attendance seamlessly. 

The system provides dedicated interfaces for both Administrators and Teachers with role-based access control to ensure data security and ease of use.

---

## 🌟 Key Features & Functionalities

### 1. Dashboard Overview
Provides a high-level summary of the institution's attendance data at a glance.
- **Key Metrics:** View the total number of students, total active subjects, and the institution's overall average attendance percentage.
- **At-Risk Tracking:** Quickly identify the number of students who have fallen below the minimum required attendance threshold (e.g., < 75%).

### 2. Role-Based Access Control (RBAC)
Secure authentication flow that directs users to appropriate interfaces based on their role:
- **Administrators:** Have full access to the system. They can view the dashboard, mark attendance, and fully manage student records (add, update, delete).
- **Teachers:** Have restricted access. They can view the dashboard and mark attendance for their respective subjects, but cannot modify core student records.

### 3. Student Management (Admin Only)
A dedicated module for administrators to maintain the student database.
- **View Records:** A searchable data table listing all enrolled students with their Roll Number, Name, Total Classes, Classes Attended, and current Attendance Percentage.
- **Dynamic Badges:** Visual indicators highlight if a student's attendance is healthy (Green), border-line (Yellow), or critical (Red).
- **Edit & Delete:** Full CRUD capabilities to manage individual records.
- **Bulk Upload:** Upload multiple students at once using a CSV file.
- **Export Reports:** Download student attendance records to CSV for offline analysis.
- **Student Profiles:** Detailed profile views showing personal information, recent attendance history, and past leave requests.

### 4. Smart Attendance Marking
A streamlined, error-free interface for teachers to record daily attendance.
- **Subject & Date Selection:** Teachers select the subject they are teaching and the specific date of the class.
- **Quick Toggles:** Attendance can be marked rapidly using "Present", "Late", or "Absent" toggle buttons for each student.
- **Bulk Saving:** Once all students are marked, a single "Save Attendance" button commits the records to the database securely.
- **Export Daily Records:** Teachers can export the daily attendance sheet for a specific subject to CSV.

### 5. Leave Requests Management
A centralized system to handle student absences.
- **Review Requests:** View all pending leave requests, complete with dates, reasons, and type.
- **Approve/Reject:** Easy one-click approval or rejection of leave requests.

## 🛠 Technology Stack

- **Frontend:** React (powered by Vite), React Router DOM for client-side routing, Lucide React for modern iconography, and custom Vanilla CSS for a premium, glassmorphism-inspired aesthetic.
- **Backend:** Node.js, Express.js.
- **Database:** MySQL.
- **Authentication:** JSON Web Tokens (JWT) & bcrypt for password hashing.

## 🚀 Setup & Installation Guide

Follow these steps to run the application on your local machine.

### Prerequisites
1. **Node.js**: Ensure Node.js (v14 or higher) is installed.
2. **MySQL Database**: Ensure you have a local MySQL server running (via XAMPP, WAMP, or standalone MySQL).

### 1. Backend Setup

1. Open your terminal and navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Initialize the Database with dummy data:
   ```bash
   npm run setup-db
   ```
   *(Note: Ensure your local MySQL server is actively running on port 3306 before running this command).*
4. Start the backend server:
   ```bash
   npm start
   ```
   The backend API will run on `http://localhost:5000`.

### 2. Frontend Setup

1. Open a **new, separate terminal window** and navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open the link provided in the terminal (usually `http://localhost:5173`) in your browser.
